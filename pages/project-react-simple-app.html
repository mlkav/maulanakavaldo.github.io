Details

Simple App is a web application developed using React technology to provide user authentication services as well as features to display user details.

Fitur:

  • Login: Users can log in to the application by entering valid credentials. Upon successful login, users will be directed to the main dashboard.
  • Register: Users can create a new account by providing the required information. The registration process is straightforward and easy to follow.
  • Log Out: Users can easily log out of their account at any time to end their session.
  • User Details: Once logged in, users can view their account details, including relevantpersonal information.
